# Ostertoaster Timeline Looper

Most loopers trap your audio inside a plugin. This one puts it on the timeline.

Place red clips to record and green clips to play back. Hit play, and as the playhead moves through them, the recordings land in the arrange view as real audio items so you can edit and arrange them afterward. There's a lot more to it, but the best way to find out is to try it 😉

## Install

1. Install [ReaImGui](https://forum.cockos.com/showthread.php?t=250419) if you don't have it
2. Extensions > ReaPack > Import repositories
3. Paste: `https://raw.githubusercontent.com/bosterho/Scheduled-Looper/main/index.xml`
4. Extensions > ReaPack > Browse packages > find "Ostertoaster Timeline Looper" > Install
5. The script appears under Actions > "Ostertoaster Timeline Looper"
